# Operations & Development Examples

Discover how to build AI agents for operations automation, development workflows, and team productivity.

## Meeting Management

### Weekly Meeting Agenda Creator

**Agent**: Weekly Meeting Agenda @Kevin_24

**Schedule**: Every Tuesday at 07:00

**Description**: Go through the notes from last week "company-weekly-meeting" in my OneNote plus fetch all my tasks that are active for this week from Microsoft Planner. Filter all emails that ticked "Yes" Create this week's meeting agenda from this info and draft an email on Outlook.

```bash
curl -X POST "https://api.incredible.com/v1/agents" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "name": "Weekly Meeting Agenda Creator",
    "description": "Go through the notes from last week 'company-weekly-meeting' in my OneNote plus fetch all my tasks that are active for this week from Microsoft Planner. Filter all emails that ticked 'Yes' Create this week's meeting agenda from this info and draft an email on Outlook",
    "schedule": "every Tuesday at 07:00",
    "user_id": "your_user_id",
    "integrations": ["onenote", "microsoft_planner", "outlook"],
    "prompt": "Read last week's meeting notes from OneNote, fetch active tasks from Microsoft Planner, filter relevant emails, and create a comprehensive meeting agenda with action items, then draft an email in Outlook"
  }'
```
