Use this to send a simple message and get a complete response.

<CodeGroup>
  ```bash Shell
curl -X POST "https://api.incredible.one/v1/chat-completion" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "small-1",
    "stream": false,
    "messages": [
      { "role": "user", "content": "Hello! Can you say hi back to me?" }
    ]
  }'
  ```

  ```python Python
import requests

response = requests.post(
    "https://api.incredible.one/v1/chat-completion",
    json={
        "model": "small-1",
        "stream": False,
        "messages": [
            {"role": "user", "content": "Hello! Can you say hi back to me?"}
        ]
    }
)
print(response.json())
  ```

  ```javascript JavaScript
const response = await fetch("https://api.incredible.one/v1/chat-completion", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "small-1",
    stream: false,
    messages: [
      { role: "user", content: "Hello! Can you say hi back to me?" }
    ]
  })
});
const data = await response.json();
console.log(data);
  ```
</CodeGroup>

## Add a system instruction

Guide the assistant with a system message and adjust generation controls:

```json
{
  "model": "small-1",
  "stream": false,
  "messages": [
    { "role": "system", "content": "You are a concise assistant." },
    { "role": "user", "content": "Summarize the Incredible API in one sentence." }
  ],
  "temperature": 0.4,
  "top_p": 0.9,
  "max_output_tokens": 120
}
```
