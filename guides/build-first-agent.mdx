---
title: "Building Your First Agent"
description: "Step-by-step tutorial"
---

1) Get an API key from your dashboard
2) Connect an integration (e.g., Gmail)
```bash
curl -X POST "https://api.incredible.one/v1/integrations/gmail/connect" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"your_user_id","callback_url":"https://your-app.com/callback"}'
```
3) Execute a feature
```bash
curl -X POST "https://api.incredible.one/v1/integrations/gmail/execute" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "your_user_id",
    "feature_name": "gmail_send_email",
    "inputs": {"recipient_email": "user@example.com", "body": "Hello from Incredible API!"}
  }'
```
