Here are the key practices that will help you build reliable applications with the Incredible API:

## üöÄ Performance

### 1. **Batch Related Requests**

Instead of multiple API calls, combine related requests:

<Tabs>
  <Tab title="cURL">
```bash
# ‚ùå Multiple calls
curl -d '{"messages": [{"role": "user", "content": "Get user 1"}]}' ...
curl -d '{"messages": [{"role": "user", "content": "Get user 2"}]}' ...

# ‚úÖ Single call
curl -d '{"messages": [{"role": "user", "content": "Get users 1 and 2"}]}' ...
```
  </Tab>
  <Tab title="Python">
```python
# ‚ùå Multiple calls
response1 = requests.post(url, json={"messages": [{"role": "user", "content": "Get user 1"}]})
response2 = requests.post(url, json={"messages": [{"role": "user", "content": "Get user 2"}]})

# ‚úÖ Single call
response = requests.post(url, json={"messages": [{"role": "user", "content": "Get users 1 and 2"}]})
```
  </Tab>
</Tabs>

### 2. **Use Streaming for Long Responses**

Enable streaming for better user experience:

<Tabs>
  <Tab title="cURL">
```bash
curl -d '{"stream": true, "messages": [{"role": "user", "content": "Generate a long report"}]}' ...
```
  </Tab>
  <Tab title="Python">
```python
response = requests.post(url, json={
    "stream": True, 
    "messages": [{"role": "user", "content": "Generate a long report"}]
}, stream=True)
```
  </Tab>
</Tabs>

### 3. **Never Hardcode Credentials**

<Tabs>
  <Tab title="cURL">
```bash
# ‚ùå Don't do this
api_key="inc_abc123..."

# ‚úÖ Use environment variables
api_key="$INCREDIBLE_API_KEY"
```
  </Tab>
  <Tab title="Python">
```python
# ‚ùå Don't do this
api_key = "inc_abc123..."

# ‚úÖ Use environment variables
import os
api_key = os.getenv("INCREDIBLE_API_KEY")
```
  </Tab>
</Tabs>

### 4. **Validate User Input**

Check prompts before sending to the API:

<Tabs>
  <Tab title="Python">
```python
def validate_prompt(prompt):
    if len(prompt) > 10000:
        return "Prompt too long"
    if "<script>" in prompt.lower():
        return "Invalid content"
    return None
```
  </Tab>
  <Tab title="JavaScript">
```javascript
function validatePrompt(prompt) {
    if (prompt.length > 10000) {
        return "Prompt too long";
    }
    if (prompt.toLowerCase().includes("<script>")) {
        return "Invalid content";
    }
    return null;
}
```
  </Tab>
</Tabs>

## üîÑ Reliability

### 5. **Handle Errors Gracefully**

<Tabs>
  <Tab title="Python">
```python
try:
    response = call_incredible_api(prompt)
    return response
except Exception as e:
    print(f"API call failed: {e}")
    return fallback_response()
```
  </Tab>
  <Tab title="JavaScript">
```javascript
try {
    const response = await callIncredibleApi(prompt);
    return response;
} catch (error) {
    console.error(`API call failed: ${error}`);
    return fallbackResponse();
}
```
  </Tab>
</Tabs>

### 6. **Implement Retries**

<Tabs>
  <Tab title="Python">
```python
def call_with_retry(prompt, max_retries=3):
    for attempt in range(max_retries):
        try:
            return call_incredible_api(prompt)
        except Exception as e:
            if attempt == max_retries - 1:
                raise e
            time.sleep(2 ** attempt)  # Exponential backoff
```
  </Tab>
  <Tab title="JavaScript">
```javascript
async function callWithRetry(prompt, maxRetries = 3) {
    for (let attempt = 0; attempt < maxRetries; attempt++) {
        try {
            return await callIncredibleApi(prompt);
        } catch (error) {
            if (attempt === maxRetries - 1) {
                throw error;
            }
            await new Promise(resolve => setTimeout(resolve, 2 ** attempt * 1000));
        }
    }
}
```
  </Tab>
</Tabs>

## üìä Monitoring

### 7. **Log API Calls**

Track usage and errors:

<Tabs>
  <Tab title="Python">
```python
import logging

def log_api_call(prompt, response, duration):
    logging.info(f"API call: {len(prompt)} chars, {duration:.2f}s, success")
```
  </Tab>
  <Tab title="JavaScript">
```javascript
function logApiCall(prompt, response, duration) {
    console.log(`API call: ${prompt.length} chars, ${duration.toFixed(2)}s, success`);
}
```
  </Tab>
</Tabs>
