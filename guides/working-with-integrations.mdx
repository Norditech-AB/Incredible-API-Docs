---
title: "Working with Integrations"
description: "Connection and usage patterns"
---

1) List integrations
```bash
curl -X GET "https://api.incredible.one/v1/integrations"
```
2) Get details
```bash
curl -X GET "https://api.incredible.one/v1/integrations/gmail"
```
3) Connect (OAuth/API key)
```bash
curl -X POST "https://api.incredible.one/v1/integrations/gmail/connect" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"your_user_id","callback_url":"https://your-app.com/callback"}'
```
