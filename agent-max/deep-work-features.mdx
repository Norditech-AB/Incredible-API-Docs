<Accordion title="How it works">
  | Aspect | Implementation |
  |--------|---------------|
  | **Mechanism** | Input validation against available context; output verification before committing |

## ‚ö° Fast Retries

**Second chances are cheap.**

When something fails, Agent MAX doesn't start over. It retries from cache‚Äîusing the successful steps already completed. Result: retries cost 80% less.

<Accordion title="How it works">
  | Aspect | Implementation |
  |--------|---------------|
  | **Mechanism** | Context caching; incremental retry |
  | **Result** | 80% lower retry costs |
  | **Guarantee** | Mistakes don't compound |
  
  **Technical details:**
  - Successful action results are cached
  - Failed actions retry with cached context
  - Cache persistence configurable per task type
</Accordion>

```
Traditional retry: Step 1 ‚Üí Step 2 ‚Üí Step 3 ‚Üí FAIL ‚Üí Step 1 ‚Üí Step 2 ‚Üí Step 3
Agent MAX retry:   Step 1 ‚Üí Step 2 ‚Üí Step 3 ‚Üí FAIL ‚Üí [cached] ‚Üí [cached] ‚Üí Step 3
```
