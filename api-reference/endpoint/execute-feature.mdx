---
title: 'Execute Integration Feature'
openapi: 'POST /v1/integrations/{integration_id}/execute'
---

Execute a specific feature from an integration. This endpoint is typically used by the agentic model to perform actions through connected integrations.

<ParamField header="integration_id" type="string" required>
  The unique identifier of the integration to execute the feature from
</ParamField>

<RequestExample>

```json
{
  "user_id": "your_user_id",
  "feature_name": "gmail_send_email",
  "inputs": {
    "recipient_email": "user@example.com",
    "body": "Hello from Incredible API!"
  }
}
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "result": {
    "message_id": "abc123...",
    "status": "sent"
  },
  "integration_id": "gmail",
  "feature_name": "gmail_send_email",
  "user_id": "your_user_id"
}
```

</ResponseExample>

<ResponseExample>

```json Response
{
  "error": "Feature not found",
  "available_features": ["gmail_send_email", "gmail_get_emails"]
}
```

</ResponseExample>
