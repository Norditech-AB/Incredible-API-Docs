---
title: "Get Integration Details"
openapi: "GET /v1/integrations/{integration_id}"
---

See what an integration can do before you connect.

## Why use this?
- Discover available features and required parameters
- Confirm authentication method and setup needs
- Show name, logo, and description in your UI

## Path Parameters

The `integration_id` should be the unique identifier of the integration you want to retrieve details for.
