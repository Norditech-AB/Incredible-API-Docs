openapi: "POST /v1/generate-image"
---

## Examples

<CodeGroup>

```python Python SDK
from incredible_python import Incredible

client = Incredible()

response = client.generate_image(
    prompt="A serene mountain landscape at sunset, digital art style",
    aspect_ratio="16:9",
    output_format="jpeg"
)

print("Image URL:", response.image_url)
print("Revised prompt:", response.revised_prompt)
```

```ts TypeScript SDK
import { IncredibleClient } from '@incredible-ai/sdk';

const client = new IncredibleClient();

const response = await client.images({
  prompt: 'A serene mountain landscape at sunset, digital art style',
  aspect_ratio: '16:9'
});

if (response.success) {
  console.log('Image URL:', response.image_url);
  console.log('Seed:', response.seed);
}
```

```bash cURL
curl -X POST "https://api.incredible.one/v1/generate-image" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene mountain landscape at sunset, digital art style",
    "aspect_ratio": "16:9",
    "output_format": "jpeg"
  }'
```

</CodeGroup>

## Parameters

- **prompt** (required) — Text description of the image
- **aspect_ratio** (optional) — Image dimensions: `1:1`, `16:9`, `9:16`, `4:3`, `3:4`, `21:9`, `9:21` (default: `16:9`)
- **output_format** (optional) — `jpeg` or `png` (default: `jpeg`)
- **seed** (optional) — Random seed for reproducibility
- **prompt_upsampling** (optional) — Enhance prompt automatically (default: `false`)
- **safety_tolerance** (optional) — Content moderation level 0-6 (default: `2`)

## Response

```json
{
  "success": true,
  "image_url": "https://storage.googleapis.com/...",
  "request_id": "abc123",
  "seed": 42,
  "revised_prompt": "Enhanced version of your prompt"
}
```

**Note:** Image URLs expire after 1 hour. Download and save images immediately.
