title: "Deep Research"

openapi: "POST /v1/deep-research"
---

## Examples

<CodeGroup>

```python Python SDK
from incredible_python import Incredible

client = Incredible()

response = client.deep_research(
    query="Comprehensive analysis of renewable energy trends in 2024",
    depth="standard"  # Options: "quick", "standard", "thorough"
)

print("Report:", response.report)
print("\nKey Findings:")
for finding in response.key_findings:
    print(f"- {finding}")
```

```ts TypeScript SDK
import { IncredibleClient } from '@incredible-ai/sdk';

const client = new IncredibleClient();

const response = await client.research.deepResearch({
  instructions: 'Comprehensive analysis of renewable energy trends in 2024'
});

if (response.success) {
  console.log('Research completed!');
  console.log('Report:', response.report || response.content);
  console.log('Citations:', response.citations?.length || 0);
}
```

```bash cURL
curl -X POST "https://api.incredible.one/v1/deep-research" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Comprehensive analysis of renewable energy trends in 2024",
    "depth": "standard"
  }'
```

</CodeGroup>

## Parameters

- **query** (required) — Research topic or question
- **depth** (optional) — Research depth: "quick", "standard", or "thorough"
- **max_sources** (optional) — Maximum sources to consult
- **response_format** (optional) — JSON schema for structured output

## Response

```json
{
  "success": true,
  "report": "Comprehensive research report...",
  "key_findings": [
    "Finding 1",
    "Finding 2"
  ],
  "sources": [
    {
      "title": "Source Title",
      "url": "https://example.com",
      "relevance": "high"
    }
  ]
}
```
