title: "Generate Video"

openapi: "POST /v1/generate-video"
---

## Examples

<CodeGroup>

```python Python SDK
from incredible_python import Incredible
import base64

client = Incredible()

response = client.generate_video(
    prompt="A golden retriever playing in a sunny park",
    size="1280x720"
)

# Extract and save video
video_data = response.video_url.split(',')[1]
with open('generated_video.mp4', 'wb') as f:
    f.write(base64.b64decode(video_data))

print("Video saved!")
```

```ts TypeScript SDK
import { IncredibleClient } from '@incredible-ai/sdk';
import fs from 'fs';

const client = new IncredibleClient();

const response = await client.videos({
  prompt: 'A golden retriever playing in a sunny park',
  size: '1280x720'
});

if (response.success && response.video_url) {
  // Extract and save video
  const base64Data = response.video_url.split(',')[1];
  const videoBuffer = Buffer.from(base64Data, 'base64');
  fs.writeFileSync('generated_video.mp4', videoBuffer);
  console.log('Video saved!');
}
```

```bash cURL
curl -X POST "https://api.incredible.one/v1/generate-video" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A golden retriever playing in a sunny park",
    "size": "1280x720"
  }'
```

</CodeGroup>

## Parameters

- **prompt** (required) — Text description of the video
- **size** (optional) — Video dimensions: `1280x720`, `720x1280`, `1920x1080`, `1080x1920`, `1024x1024` (default: `1280x720`)
- **input_reference** (optional) — Base64-encoded reference image for image-to-video

## Response

```json
{
  "success": true,
  "video_id": "operation_abc123",
  "video_url": "data:video/mp4;base64,AAAIGZ0...",
  "status": "completed",
  "duration": 8
}
```

**Note:** Generation takes 1-10 minutes. Videos are 8 seconds long. The `video_url` is a base64 data URI—extract and decode to save as MP4 file.
