## Examples

<CodeGroup>

```python Python SDK
from incredible_python import Incredible

client = Incredible()

# Simple answer
response = client.answer(query="What is the capital of France?")
print(response.answer)

# Structured output
schema = {
    "type": "object",
    "properties": {
        "sentiment": {"type": "string", "enum": ["positive", "negative", "neutral"]},
        "confidence": {"type": "number"}
    }
}

response = client.answer(
    query="Analyze this review: 'Great product, highly recommend!'",
    response_format=schema
)
print(response.data)  # {"sentiment": "positive", "confidence": 0.95}
```

```ts TypeScript SDK
import { IncredibleClient } from '@incredible-ai/sdk';

const client = new IncredibleClient();

// Simple answer
const response = await client.answer({
  query: 'What is the capital of France?'
});
console.log(response.answer);

// Structured output
const schema = {
  type: 'object',
  properties: {
    sentiment: { type: 'string', enum: ['positive', 'negative', 'neutral'] },
    confidence: { type: 'number' }
  }
};

const structured = await client.answer({
  query: 'Analyze sentiment: "Great product!"',
  responseFormat: schema
});
console.log(structured.data);
```

```python OpenAI SDK
from openai import OpenAI

client = OpenAI(
    base_url="https://api.incredible.one/v1",
    api_key="YOUR_API_KEY"
)

response = client.chat.completions.create(
    model="small-1",
    messages=[
        {"role": "user", "content": "What is the capital of France?"}

```

```bash cURL
curl -X POST "https://api.incredible.one/v1/answer" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What is the capital of France?"
  }'
```

</CodeGroup>

## Parameters

- **query** (required) — The question to answer
- **response_format** (optional) — JSON schema for structured output
- **file_ids** (optional) — List of file IDs to include with the query
- **stream** (optional) — Enable streaming response

## Response

**Simple mode:**
```json
{
  "success": true,
  "answer": "The capital of France is Paris."
}
```

**Structured mode:**
```json
{
  "success": true,
  "data": {
    "sentiment": "positive",
    "confidence": 0.95
  }
}
```

## Using Files

Attach files to your query using `file_ids`. Files are automatically processed and their content is included in the query context.

```python
# Upload file first (see Files API), then use file_id
response = client.answer(
    query="Summarize this document",
    file_ids=["file-abc123", "file-def456"]
)
```

<Info>
**Upload Once, Reuse Forever**

Upload files using the [Files API](/api-reference/files/overview) once, then reuse the `file_id` across unlimited requests. No need to re-upload the same file.
</Info>
