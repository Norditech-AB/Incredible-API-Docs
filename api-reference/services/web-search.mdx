## Examples

<CodeGroup>

```python Python SDK
from incredible_python import Incredible

client = Incredible()

response = client.web_search(
    query="Latest developments in quantum computing 2024",
    max_results=5
)

print("Answer:", response.answer)
print("\nSources:")
for source in response.sources:
    print(f"- {source['title']}: {source['url']}")
```

```ts TypeScript SDK
import { IncredibleClient } from '@incredible-ai/sdk';

const client = new IncredibleClient();

const response = await client.research({
  query: 'Latest developments in quantum computing 2024',
  num_results: 5
});

if (response.success) {
  console.log('Answer:', response.answer);
  console.log('\nSources:');
  response.results?.forEach(result => {
    console.log(`- ${result.title}: ${result.url}`);
  });
}
```

```bash cURL
curl -X POST "https://api.incredible.one/v1/web-search" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "Latest developments in quantum computing 2024",
    "max_results": 5
  }'
```

</CodeGroup>

## Parameters

- **query** (required) — Search query
- **max_results** (optional) — Number of results to return
- **include_domains** (optional) — Filter to specific domains
- **exclude_domains** (optional) — Exclude specific domains
- **start_published_date** (optional) — Filter by date range
- **end_published_date** (optional) — Filter by date range
- **category** (optional) — Filter by category (company, research paper, news, github, tweet, pdf)

