Welcome to **Incredible**! This guide will get you up and running with the Incredible API in under 10 minutes.

## üöÄ Prerequisites

Before you begin, you'll need:
- An Incredible API key ([Get one here](https://www.incredible.one))
- `curl` installed (or use any HTTP client)
- Basic familiarity with APIs

## üîë Step 1: Get Your API Key

1. Visit [incredible.one](https://www.incredible.one)
2. Sign up for an account
3. Navigate to your dashboard
4. Copy your API key (it starts with `inc_`)

Set your API key as an environment variable:

```bash

# Windows (Command Prompt)

```

## üì° Step 3: Make Your First API Call

### Basic Chat Request

Run this cURL command to send your first request:

```bash
curl -X POST "https://api.incredible.one/v1/chat-completion" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $INCREDIBLE_API_KEY" \
  -d '{
    "model": "small-1",
    "messages": [
      { "role": "user", "content": "Hello, Incredible! What can you do?" }
    ]
  }'
```

**Expected Response:**

```json
{
  "result": {
    "thinking": "The user is greeting me and asking about my capabilities. I should respond politely and explain what I can do as an Incredible AI assistant.",
    "response": [
      {
        "content": "Hello! I'm an AI assistant powered by Incredible, and I'm here to help you. I have access to various tools that allow me to run functions, perform analysis, and execute code. I can help with mathematical operations, API integrations, data processing, and much more. What would you like to work on today?",
        "role": "assistant"
      }
    ]
  }
}
```

### Understanding the Response

The response includes:
- **`thinking`**: The model's reasoning process before responding
- **`response`**: The actual assistant message(s)
- **`role`**: Always "assistant" for AI responses

## üîÑ Step 4: Try Streaming

Enable streaming to see responses in real-time:

```bash
curl -X POST "https://api.incredible.one/v1/chat-completion" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $INCREDIBLE_API_KEY" \
  -d '{
    "model": "small-1",
    "stream": true,
    "messages": [
      { "role": "user", "content": "Count from 1 to 5 slowly." }

  }'
```

## üö® Troubleshooting

### Common Issues

**401 Unauthorized:**
- Check that your API key is correct
- Ensure the Authorization header includes "Bearer "
- Verify your API key hasn't expired

**400 Bad Request:**
- Check your JSON syntax
- Ensure all required fields are present
- Verify the model name is correct

**Rate Limiting:**
- If you get rate limited, wait a moment and try again
- Check your dashboard for current usage
