Welcome to **Incredible**! This guide will get you up and running in under 3 minutes.

## üöÄ Step 1: Make Your First API Call

The Incredible API works without authentication for basic usage. Here's your first request:

<CodeGroup>
  ```bash Shell
curl -X POST "https://api.incredible.one/v1/chat-completion" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "small-1",
    "messages": [
      { "role": "user", "content": "Hello, Incredible!" }
    ]
  }'
  ```

  ```python Python
import requests

response = requests.post(
    "https://api.incredible.one/v1/chat-completion",
    json={
        "model": "small-1",
        "messages": [
            {"role": "user", "content": "Hello, Incredible!"}
        ]
    }
)
print(response.json())
  ```

  ```javascript JavaScript
const response = await fetch("https://api.incredible.one/v1/chat-completion", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "small-1",
    messages: [
      { role: "user", content: "Hello, Incredible!" }
    ]
  })
});
const data = await response.json();
console.log(data);
  ```
</CodeGroup>

**Response:**
```json
{
  "result": {
    "thinking": "The user is greeting me...",
    "response": [
      {
        "content": "Hello! I'm here to help you...",
        "role": "assistant"
      }
    ]
  }
}
```

## üîÑ Step 2: Try Streaming

Streaming is enabled by default, but you can explicitly control it:

<CodeGroup>
  ```bash Shell
curl -X POST "https://api.incredible.one/v1/chat-completion" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "small-1",
    "stream": true,
    "messages": [
      { "role": "user", "content": "Count to 3" }
    ]
  }'
  ```

  ```python Python
import requests

response = requests.post(
    "https://api.incredible.one/v1/chat-completion",
    json={
        "model": "small-1",
        "stream": True,
        "messages": [
            {"role": "user", "content": "Count to 3"}
        ]
    },
    stream=True
)

for line in response.iter_lines():
    if line:
        print(line.decode('utf-8'))
  ```

  ```javascript JavaScript
const response = await fetch("https://api.incredible.one/v1/chat-completion", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "small-1",
    stream: true,
    messages: [
      { role: "user", content: "Count to 3" }
    ]
  })
});

const reader = response.body.getReader();
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  console.log(new TextDecoder().decode(value));
}
  ```
</CodeGroup>
